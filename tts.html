🗣️ Локальный TTS — озвучка текста

Веб-приложение использует системные голоса Windows (SAPI / Microsoft TTS) через Web Speech API. Работает офлайн, без отправки текста на сторонние серверы.

⚠ Выбранный голос — Microsoft SAPI: параметр «Тон» (pitch) этим движком обычно игнорируется. Для смены тона используйте кнопку «💾 Сохранить в файл» — записанное аудио можно затем сместить по высоте через постобработку (см. README).

Горячие клавиши и подсказки

Ctrl + Enter — озвучить · Esc — остановить.

Размер фрагмента: 80–10 000 символов. Большие фрагменты звучат естественнее, но движки SAPI могут обрываться. Рекомендованный диапазон: 200–2 000.

Доп. рекомендации

В Windows: «Параметры → Время и язык → Речь → Управление голосами» — добавьте Microsoft Irina / Pavel или установите дополнительные пакеты SAPI.

Для офлайн-нейросетевого качества используйте серверный движок (Piper, Coqui XTTS) — см. README.
