🗣️ Локальный TTS — озвучка текста

Веб-приложение использует системные голоса Windows (SAPI / Microsoft TTS) через Web Speech API. Работает офлайн, без отправки текста на сторонние серверы.

Текст для озвучивания

0 символов · 0 фрагментов · оценка: ~0 с
Готов к работе.
Очередь воспроизведения
Горячие клавиши и подсказки

Ctrl + Enter — озвучить · Esc — остановить.

Длинные тексты автоматически делятся на фрагменты до 200 символов по границам предложений, что снижает задержки и обходит ограничения движков Windows.

Сохранение в файл реализовано через запись аудио вкладки (Chrome / Edge): после нажатия «Сохранить» появится системный диалог — выберите эту вкладку и обязательно отметьте «Поделиться звуком».

Доп. рекомендации

В Windows для русского языка установите «Параметры → Время и язык → Речь → Управление голосами» — добавьте Microsoft Irina / Pavel или загрузите дополнительные пакеты SAPI.

Для офлайн-нейросетевого качества используйте серверный движок (Piper, Coqui XTTS) — см. README проекта.
